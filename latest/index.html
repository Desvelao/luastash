Module luastash

LuaStash Module This module provides functionality to process data through a pipeline of generators and outputs.

It includes logging and error handling mechanisms.

Functions

__call (config, processors, ctx, utils)
Runs the stash pipeline. This function processes data through a series of generators, filters and outputs.

Parameters:

  • config table Pipeline configuration or file path
  • processors StashProcessors Inputs, filters and outputs processors
  • ctx table Context object passed to processors
  • utils table Utility functions and logger

Returns:

    nil

Usage:

    local luastash = require("luastash")
luastash(config, processors, ctx, utils)
create_logger (options)
Creates a new logger instance.

Parameters:

  • options A table of options to configure the logger.

Returns:

    A new Logger instance.

Usage:

    local luastash = require("luastash")
local logger = luastash.create_logger({name="mylogger", level="debug"})

Tables

StashProcessors
Define the inputs, filters and outputs processors. This table should contain functions for each processor type used in the stash pipeline.

Fields:

  • inputs Defines input processor functions.
  • filters Defines filter processor functions.
  • outputs Defines output processor functions.
Stash
A metatable that provides callable and indexable behavior. This table allows calling run_stash directly and provides a method to create a logger.

Fields:

  • __call Calls the run_stash function with the provided arguments.
  • __index Contains utility methods, such as create_logger and INPUT_CONTINUE_SIGNAL.
